CAP3785 — Data Analytics and Visualization

Ethics, Data Types, and Data Sources in Analytics

This module explores the ethical foundations of data analytics, including privacy, fairness, transparency, and bias. Learners will examine different data types and sources while developing a critical understanding of responsible analytical practice.
