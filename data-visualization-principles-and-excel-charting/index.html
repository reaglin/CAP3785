CAP3785 — Data Analytics and Visualization

Data Visualization Principles and Excel Charting

This module introduces core principles of effective data visualization and hands-on Excel charting techniques. Learners will explore how to select appropriate chart types, apply visual hierarchy, use color purposefully, and avoid representations that mislead or distort data — essential skills for communicating insights clearly and honestly.
