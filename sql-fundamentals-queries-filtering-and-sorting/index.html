CAP3785 — Data Analytics and Visualization

SQL Fundamentals: Queries, Filtering, and Sorting

This module introduces the core skills needed to interact with relational databases using SQL. Learners will explore database structure, write SELECT queries, apply WHERE filters to narrow result sets, and control output order with ORDER BY — building a solid foundation for data retrieval and analysis tasks.
