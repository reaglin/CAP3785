CAP3785 — Data Analytics and Visualization

SQL Fundamentals: Queries, Filtering, and Sorting

This module introduces the core skills of querying relational databases with SQL. Learners will explore how to retrieve, filter, sort, and limit data using SELECT, WHERE, ORDER BY, and LIMIT, and will practice translating real-world business questions into working SQL queries.
