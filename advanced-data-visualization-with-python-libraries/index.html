CAP3785 — Data Analytics and Visualization

Advanced Data Visualization with Python Libraries

This module introduces powerful Python visualization tools — primarily Matplotlib and Seaborn — guiding learners from foundational concepts through core chart types, advanced multivariate techniques, publication-quality styling, data storytelling, and interactive figure building.
