CAP3785 — Data Analytics and Visualization

Pivot Tables, Dashboards, and Power Query in Excel

This module guides learners through the essential Excel toolkit for data analysis and reporting — from building and refining pivot tables to designing interactive dashboards and automating data preparation with Power Query. Use the artifacts below to read, study, and assess your progress.
