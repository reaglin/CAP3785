CAP3785 — Data Analytics and Visualization

Module 3 — Data Quality Assessment and Data Cleaning Fundamentals

This module introduces the principles and practice of evaluating and improving data quality. Topics range from understanding core quality dimensions and identifying common issues — such as missing values, duplicates, and inconsistencies — to applying systematic cleaning strategies that produce analysis-ready datasets. Use the resources below to work through readings, practice activities, and assessments at your own pace.
