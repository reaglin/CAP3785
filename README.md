# CAP3785 - Data Analytics and Visualization

This course gives a broad overview of the various aspects of data analytics and visualizations. Students will learn ways of accessing data from various sources such as web APIs and repositories, techniques of cleaning data and organizing data for analysis, using analytical methods to solve real-world problems, and create visualizations to aid the interpretation of analysis results. Students will have hands on training using relevant programming languages, and analytics and visualization tools

## Modules

1. 1 - Introduction to Data Analytics and the Analytics Lifecycle
2. 2 - Ethics, Data Types, and Data Sources in Analytics
3. 3 - Data Quality Assessment and Data Cleaning Fundamentals
4. 4 - Database Basics and Relational Data Concepts
5. Advanced Excel Functions and Statistical Tools
6. Pivot Tables, Dashboards, and Power Query in Excel
7. Data Visualization Principles and Excel Charting
8. SQL Fundamentals: Queries, Filtering, and Sorting
9. SQL Joins, Relationships, and Advanced Query Techniques
10. Python Basics for Data Analysis
11. Data Manipulation and Cleaning with NumPy and Pandas
12. Descriptive Statistics and Probability Distributions
13. Hypothesis Testing, Correlation, and Regression Analysis
14. Advanced Data Visualization with Python Libraries
15. Introduction to Machine Learning for Business Analytics
