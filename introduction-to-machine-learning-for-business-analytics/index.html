CAP3785 — Data Analytics and Visualization

Introduction to Machine Learning for Business Analytics

This module introduces foundational machine learning concepts within a business analytics context. Learners will explore supervised and unsupervised learning, key algorithms, and model evaluation techniques — equipping them to apply ML thinking to real-world data challenges.
