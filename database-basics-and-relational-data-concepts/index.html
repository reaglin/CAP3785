CAP3785 — Data Analytics and Visualization

Database Basics and Relational Data Concepts

This module introduces the foundational principles of databases and relational data modeling. Learners will explore how data is structured using tables, rows, columns, and keys; how relational databases differ from flat-file approaches; and how schemas organize and connect information across a system.
