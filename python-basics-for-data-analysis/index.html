CAP3785 — Data Analytics and Visualization

Python Basics for Data Analysis

This module introduces the foundational Python skills required for data analytics work. Learners will configure a professional development environment, explore Python's core data types and structures, apply control-flow logic, write reusable functions, handle file input/output, and assemble complete data analysis scripts from scratch.
